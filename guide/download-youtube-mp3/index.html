Как сохранить звук с YouTube (MP3)

Без мутных «конвертеров». Два рабочих хода: бот в Telegram (YouTube+ВК) и ru-video.ru — если исходник лежит в ВК. Плюс — как корректно конвертировать и где искать файлы.

Рабочие способы (пошагово)

Способ 1. Telegram-бот @SaveFromVkBot

YouTube + ВК
  1. Откройте @SaveFromVkBot (iPhone/Android/ПК).
  2. Скопируйте ссылку на ролик YouTube и отправьте боту.
  3. Если бот предложит «audio/mp3» — выберите его. Если даёт только видео — скачайте видео-файл (MP4/WEBM).
  4. Готовый MP3 сохраните (или конвертируйте видео в MP3 локально — см. раздел ниже).

Плюсы: быстро, без регистрации и лишних сайтов. Минусы: нужен Telegram; для отдельных роликов аудио может быть недоступно — тогда конвертация.

Способ 2. Если исходник в ВК: ru-video.ru

Только ВК
  1. Скопируйте ссылку на видео во ВКонтакте с нужным треком.
  2. Откройте ru-video.ru в браузере → вставьте ссылку → «Скачать» → сохраните видео-файл.
  3. Преобразуйте файл в MP3 локально (см. «Конвертация» ниже) и добавьте теги/обложку.

Плюсы: стабильно для ВК. Минусы: сайт не работает с YouTube напрямую; возможна реклама.

Конвертация MP4 → MP3 (без лишних рисков)

ПК/ноутбук (Windows/macOS)

  1. Установите VLC или другой доверенный плеер/конвертер.
  2. VLC → «Медиа» → «Преобразовать/Сохранить» → добавьте видео → профиль Audio - MP3 → «Старт».
  3. Проверьте битрейт: 192–320 kbps — ок для музыки, 128 kbps — норм для подкастов.

Альтернатива: FFmpeg (продвинутым) — быстро и гибко.

iPhone / Android

  • iPhone: скачайте файл через бота → «Поделиться» → «Сохранить в Файлы». Для конвертации используйте проверенное приложение-плеер с офлайн-конвертацией (без облаков/регистраций) или перенесите на ПК и конвертируйте там.
  • Android: файл в «Download/Загрузки». Конвертируйте через офлайн-плеер/конвертер из Google Play с хорошими отзывами или на ПК.

Старайтесь избегать «онлайн-конвертеров» с агрессивной рекламой и загрузкой файлов «в облако».

ID3-теги и обложка: после конвертации добавьте название, автора и обложку в тег-редакторе (mp3tag, MusicBrainz Picard, плеер на телефоне).

Где искать файл после скачивания

ПК/ноутбук

По умолчанию — папка «Загрузки». В браузере можно открыть список загрузок и перейти к файлу.

Android

Папка Download/Загрузки в файловом менеджере. MP3 виден и в приложении «Музыка».

iPhone

Через бота: «Поделиться» → «Сохранить в Файлы» (iCloud/«На iPhone»). Для «Музыки» добавляйте через десктопный плеер/синхронизацию или слушайте в плеере с доступом к «Файлам».

Если YouTube не открывается в России

Быстрое решение (3 минуты)

  1. Откройте SAFENET-VPN в Telegram.
  2. Выберите устройство → получите WireGuard/OpenVPN конфиг.
  3. Подключитесь к NL/DE → проверьте доступ к YouTube и скорость.

Что не работает в 2025

Случайные «YouTube→MP3» сайты

Часто ломаются, навязывают софт, сливают файлы «в облако».

Расширения «в один клик»

Вылетают из магазинов браузеров, шпионят, не дружат с обновлениями YouTube.

«Через код страницы → прямой .mp3»

Неактуально: потоки фрагментированы/защищены. Потратите время — получите ноль.

Вывод

Надёжнее всего: @SaveFromVkBot (если даёт аудио) или скачайте видео и конвертируйте локально. Для ВК — дополнительно ru-video.ru.

FAQ

Какой битрейт выбрать?

Музыка — 192–320 kbps. Для подкастов хватит 128 kbps.

Формат AAC vs MP3?

AAC при том же битрейте звучит лучше, но MP3 совместимее. Выбирайте под свои плееры.

Как добавить обложку?

Через редактор тегов (mp3tag/Picard) или в мобильном плеере с поддержкой ID3.

Можно ли сразу плейлист?

Бот работает по ссылке на ролик. Для плейлиста берите треки по одному.
