Скачать видео с YouTube на iPhone

Айфон не любит прямые загрузки. Но у нас — два рабочих хода: бот в Telegram (YouTube+ВК) и ru-video.ru (только ВК). Плюс — как сохранить в «Фото».

Рабочие способы (пошагово)

Способ 1. Telegram-бот @SaveFromVkBot

YouTube + ВК
  1. Откройте @SaveFromVkBot в Telegram на iPhone.
  2. Скопируйте ссылку на видео (YouTube или ВК). Для ВК — именно на видео, не на пост.
  3. Отправьте ссылку боту → дождитесь файла/вариантов качества.
  4. Нажмите на файл → значок «Поделиться» → Сохранить видео (в «Фото») или «Сохранить в Файлы» (память устройства/iCloud).

Плюсы: быстро и без сторонних приложений. Минусы: нужен Telegram.

Способ 2. Ruvideo — ru-video.ru

Только ВК
  1. Скопируйте ссылку на видео во ВКонтакте.
  2. Откройте Safari → перейдите на ru-video.ru.
  3. Вставьте ссылку → «Скачать» → выберите качество → загрузите файл.
  4. После загрузки: значок «Поделиться» → Сохранить видео (в «Фото») или «Сохранить в Файлы».

Плюсы: без приложений и бесплатно. Минусы: только для ВК, возможна реклама.

Как сохранить в «Фото» или «Файлы»

Сохранить в «Фото» (Галерею)

  1. Нажмите на скачанный ролик → Поделиться.
  2. Выберите Сохранить видео.
  3. Откройте «Фото» → «Недавние» → проверьте файл (MP4/WEBM → iOS конвертирует для воспроизведения).

Сохранить в «Файлы»

  1. Поделиться → Сохранить в Файлы.
  2. Выберите «На iPhone» или «iCloud Drive» (папка «Downloads»/«Загрузки»).
  3. Откройте «Файлы» → найдите видео → при желании перенесите в «Фото».

Если пункт «Сохранить видео» отсутствует — файл в формате/кодеке, не поддерживаемом «Фото». Сохраните в «Файлы» и откройте через сторонний плеер.

Легально: офлайн и свои ролики

Официальный офлайн в YouTube

  1. Откройте ролик в приложении YouTube → кнопка «Скачать» (если доступно).
  2. Выберите качество → смотрите во вкладке «Загрузки».

Файл остаётся внутри приложения и не появляется в «Фото».

Скачать СВОИ ролики

  1. YouTube Studio в Safari (режим «Версия для ПК») → «Контент» → «⋯» → «Скачать».
  2. Google Takeout → «YouTube и YouTube Music» → сформируйте архив и скачайте на iPhone.

Если YouTube не открывается в России

Быстрое решение (3 минуты)

  1. Откройте бота SAFENET-VPN.
  2. Выберите iPhone → получите профиль/конфиг (WireGuard/OpenVPN).
  3. Подключитесь к NL/DE → проверьте доступ к YouTube и загрузки.

Что не работает в 2025

Такие приложения массово выпиливают/ограничивают. Вчера работало — сегодня нет.

Сайты «по ссылке — mp4» для YouTube

Часто ломаются на защищённых роликах, плюс реклама/фишинг. Для ВК — норм: ru-video.ru.

«Через код страницы»/прямые .mp4

Потоки сегментированы/шифруются — на iOS бессмысленно.
